# 健康事件追踪系统 - 快速开始

## 🚀 立即开始使用

### 1. 访问健康事件页面

在应用左侧栏点击 **"健康事件"** 链接，或访问 `/health-events` 路由。

### 2. 查看默认示例数据

页面默认加载了 4 个示例事件：
- 🟤 胃痛 （进行中）
- 🟤 感冒 （已康复）
- 🟤 头痛 （进行中）
- 🟤 腰扭伤 （已康复）

### 3. 切换视图

在页面顶部的标签栏中：
- 点击 **列表视图** 看清晰的卡片布局
- 点击 **时间轴视图** 看事件演变历程

## 📝 常见操作

### 添加新事件
```
1. 点击蓝色的"新增健康事件"按钮
2. 填写事件信息
3. 点击"创建"
```

### 编辑事件
```
1. 找到要编辑的事件卡片
2. 点击编辑（✏️）图标
3. 修改信息后点击"更新"
```

### 标记为已解决
```
1. 在进行中的事件上点击✅按钮
2. 事件自动移至"已康复"区域
```

### 删除事件
```
1. 点击删除（🗑️）图标
2. 确认删除
```

## 🔍 搜索和过滤（列表视图）

1. 在搜索框输入关键词搜索事件名称或描述
2. 点击"显示筛选"展开高级选项
3. 选择事件分类和状态进行过滤
4. 点击"重置筛选"清空所有过滤

## 💾 导出数据

点击右上角的菜单（⋮）按钮：
- **导出为 CSV**：下载 Excel 可打开的文件
- **打印**：打印事件记录

| 特性 | 描述 |
|------|------|
| 📊 统计面板 | 实时显示事件总数、进行中数、已康复数、待跟进数 |
| 🌙 深色模式 | 完全支持深色模式，护眼舒适 |

## 🏥 事件分类

系统支持 8 种医学分类：

| 分类 | 用例 |
|------|------|
| 🍽️ 消化系统 | 胃痛、腹泻、便秘等 |
| 🫁 呼吸系统 | 感冒、咳嗽、哮喘等 |
| 🧠 神经系统 | 头痛、头晕、失眠等 |
| 💪 骨科 | 腰痛、扭伤、骨折等 |
| ❤️ 心血管 | 心悸、高血压等 |
| 🩹 皮肤科 | 皮疹、湿疹等 |
| 🦠 传染病 | 流感、肺炎等 |
| 📋 其他 | 其他医学相关事件 |

## 💡 使用建议

1. **及时记录**：发生健康事件时立即记录，便于后续查询
2. **设置跟进**：进行中的事件设置下次跟进日期，系统会提醒
3. **定期回顾**：定期查看时间轴，了解健康趋势
4. **归档完成**：及时将已康复的事件标记为已解决
5. **导出备份**：定期导出数据作为备份

- 刷新页面后示例数据会重置
