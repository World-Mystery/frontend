# 健康事件追踪系统

一个功能完整、设计精美的健康事件管理界面，用于追踪和管理个人或家庭成员的健康状况。

## 功能特性

### 📊 核心功能

- **双视图模式**
  - **列表视图**：清晰展示所有健康事件，便于快速浏览和编辑
  - **时间轴视图**：时间线展示，直观了解健康历程演变

- **事件状态管理**
  - 🔴 **进行中**：需要持续跟进和治疗的状态
  - 🟢 **已康复/归档**：已经解决或康复的事件

- **详细事件追踪**
  - 事件名称、具体描述
  - 事件发生日期
  - 事件分类（消化系统、呼吸系统、神经系统等 8 个分类）
  - 下次跟进日期（进行中的事件可设置）
  - 自动计算距离下次跟进的天数

### 🎯 交互功能

- **事件操作**
  - ✏️ 编辑：修改已有事件的所有信息
  - 🗑️ 删除：永久删除不需要的事件
  - ✅ 标记解决：快速将进行中的事件标记为已康复

- **搜索和过滤**
  - 按事件名称或描述搜索
  - 按事件分类过滤
  - 按状态过滤（进行中/已康复）
  - 支持多条件组合

- **数据导出**
  - 导出为 CSV 格式，便于在其他工具中使用
  - 打印功能，支持纸质记录

### 📈 统计面板

实时显示关键指标：
- 总事件数
- 进行中的事件数
- 已康复的事件数
- 待跟进的事件数

## 使用指南

### 新增事件

1. 点击"新增健康事件"按钮
2. 填写以下信息：
   - **事件名称**（必填）：如"胃痛"、"感冒"等
   - **具体描述**（可选）：症状、诊断等详细信息
   - **事件分类**（必填）：选择对应的医学分类
   - **事件日期**（必填）：选择事件发生的日期
   - **事件状态**（必填）：
     - 选择"治疗/跟进中"：可额外设置下次跟进日期
     - 选择"已康复/归档"：直接归档
3. 点击"创建"保存事件

### 编辑事件

1. 在列表或时间轴视图中找到要编辑的事件
2. 点击编辑（✏️）按钮
3. 修改需要更新的信息
4. 点击"更新"保存更改

### 标记为已解决

1. 在进行中的事件卡片上点击"解决"（✅）按钮
2. 事件将自动移至"已康复/归档"部分

### 搜索和过滤

#### 列表视图
1. 在搜索框中输入关键词（事件名称或描述）
2. 点击"显示筛选"按钮展开高级过滤选项
3. 选择事件分类和状态进行过滤
4. 点击"重置筛选"清空所有过滤条件

#### 时间轴视图
- 所有事件按时间倒序排列（最新的在上）
- 支持通过颜色和状态徽章快速识别事件类型

### 导出和打印

1. 点击右上角的菜单（⋮）按钮
2. 选择"导出为 CSV"或"打印"
3. CSV 文件将包含所有事件的完整信息

## 界面设计

### 色彩系统

- **状态指示**：

## 数据结构

每个健康事件包含以下字段：

```typescript
interface HealthEvent {
  id: string                    // 唯一标识符
  title: string                 // 事件名称
  description: string           // 详细描述
  date: Date                    // 发生日期
  status: "active" | "recovered" // 事件状态
  nextFollowUp?: Date           // 下次跟进日期（可选）
  category: EventCategory       // 事件分类
}
```

### 支持的事件分类

- 🍽️ 消化系统（digestive）
- 🫁 呼吸系统（respiratory）
- 🧠 神经系统（neurological）
- 💪 骨科（orthopedic）
- ❤️ 心血管（cardiovascular）
- 🩹 皮肤科（dermatological）
- 🦠 传染病（infectious）
- 📋 其他（other）
